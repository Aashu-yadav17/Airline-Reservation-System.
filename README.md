# ✈️ Airline Reservation System

##  Description
This is a Java Swing based Airline Reservation System.
It allows users to view flights, book tickets, cancel bookings, and manage seat availability.

## Features
- View available flights
- Book tickets with unique ID
- Cancel tickets
- View bookings
- GUI with background image

## 🛠️ Technologies Used
- Java
- Swing (GUI)
- OOP Concepts
